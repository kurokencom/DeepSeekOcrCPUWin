# DeepSeek OCR - ポータブル Windows 版

このフォルダには、DeepSeek OCR アプリケーションのポータブル版が含まれています。

## 必要条件

- Windows 64ビット オペレーティングシステム。
- C++ランタイム（https://learn.microsoft.com/ja-jp/cpp/windows/latest-supported-vc-redist?view=msvc-170）がインストールされていること（Visual Studio Build Toolsのインストールが必要かもしれません。※VS→C++デスクトップ開発のインストール）
- Python 3.8 (またはそれ以降、但し!!!3.12以前であること!!!) がシステムにインストールされていること。
 ※インストール中に「Add Python to PATH」のチェックボックスをオンにしてください。

※以下、ライブラリインストール時に必要になる可能性があります。
- **Rust および Cargo (Rustのパッケージマネージャー) がシステムにインストールされていること。**
  - Rustの公式ウェブサイト ([https://rustup.rs/](https://rustup.rs/)) から `rustup-init.exe` をダウンロードし、指示に従ってインストールしてください。
  - インストール後、コマンドプロンプトまたはPCの再起動が必要な場合があります。

## 実行方法
1.  `run_ocr.bat` ファイルを実行します。
2.  初回実行時には、Python 仮想環境が作成され、必要なすべてのライブラリがインストールされます。これにはインターネット接続とコンピュータの速度に応じて数分かかる場合があります。しばらくお待ちください。
3.  インストール後、OCR 処理はサンプル画像 (`test.png`) に対して自動的に開始されます。
4.  セットアップが完了しているため、2回目以降の実行ははるかに高速になります。

## 独自の画像での使用方法

このツールは、以下の2つの方法で独自の画像に対して実行できます。

### オプション 1: ドラッグ＆ドロップ
画像ファイルを `run_ocr.bat` ファイルに直接ドラッグ＆ドロップします。OCR 処理がその画像に対して実行されます。

### オプション 2: コマンドライン
1.  このディレクトリでコマンドプロンプトを開きます (エクスプローラーのアドレスバーに `cmd` と入力して Enter キーを押すと開けます)。
2.  `your_image.png` をご自身の画像へのパスに置き換えて、以下のコマンドを実行します。
    ```
    run_ocr.bat --image_file your_image.png
    ```

## 出力

OCR 結果は、`DSOCR_Release` ディレクトリ内の `result` という新しいフォルダに保存されます。これには以下が含まれます。
- `result.mmd`: 抽出されたテキストを含む Markdown ファイル。
- `result_with_boxes.jpg`: 検出されたテキストの周りにバウンディングボックスが描画された元の画像。
- `images/`: ドキュメント内で見つかった特定の要素の切り抜き画像を含むフォルダ。
